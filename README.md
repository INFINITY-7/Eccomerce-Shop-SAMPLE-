# 🛒 E-Commerce Website

A modern **E-Commerce Website** built using **HTML, CSS, JavaScript, Node.js, and Express.js**. The project demonstrates the basic functionality of an online shopping platform, including product browsing, shopping cart management, and checkout.

---

# 📌 Project Overview

This project was developed as a learning project for web development and can also be used as a **college mini project or final-year project**.

The website allows users to:
- Browse products
- Add products to the cart
- View cart items
- Remove items from the cart
- Calculate total price
- Proceed to checkout
- Simulate online payment

# ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/ecommerce-website.git
```

Go inside the project folder

```
cd ecommerce-website
```

Install dependencies

```
npm install
```

Run the server

```
node server.js
```

Open browser

```
http://localhost:3000
```

# 💻 How It Works

## Home Page

Displays all available products.

Each product contains:

- Product Image
- Product Name
- Price
- Add to Cart Button

Example:

```
Nike Shoes

$50

[ Add to Cart ]
```

---

## Shopping Cart

Users can:

- View selected products
- Remove products
- View quantity
- View total price

Example:

```
Shopping Cart

Nike Shoes      $50

Backpack        $30

-------------------

Total = $80
```

---

## Checkout

The checkout page displays:

- Customer Information
- Shipping Details
- Order Summary
- Total Amount
- Payment Button

Example:

```
Name

Address

Phone Number

[ Proceed to Payment ]
```
